## Code

### Inline Code

You can print to the terminal using the `puts "hello world"` command.

### Multi line Code

#### Without highlighting

```
def hellow_world
  puts "Hello World!"
end
```

#### With highlighting

```rb
def hellow_world
  puts "Hello World!"
end
```
